# 🛒 Easy Order App

Мобильное приложение для автоматизации заказов и управления продажами, разработанное на **Flutter** с использованием **Firebase**.

## 🚀 О проекте
**Easy Order** — это современное решение для малого и среднего бизнеса (e-commerce), которое помогает пользователям легко выбирать товары, а владельцам — управлять заказами в реальном времени.

### Основные функции:
* **Авторизация и Профиль:** Вход и регистрация пользователей через Firebase Auth.
* **Каталог товаров:** Удобный просмотр товаров с категориями.
* **Управление заказами:** Быстрое оформление и отслеживание статуса заказа.
* **Локализация:** Поддержка нескольких языков (intl).
* **Кроссплатформенность:** Работает на iOS, Android и macOS.

---

## 🛠 Технологический стек
* **Framework:** [Flutter](https://flutter.dev) (Dart)
* **Backend:** [Firebase](https://firebase.google.com) (Auth, Firestore, Storage)
* **State Management:** Provider / BLoC
* **Database:** Cloud Firestore (NoSQL)
